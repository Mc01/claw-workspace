## Screenshot Best Practices

### Content
- Use **realistic data** (not "Test Chama 123")
- Show **diverse African names** for members
- Use **local stablecoins** (cKES, cNGN, cGHS)
- Display **realistic amounts** (not 0.000001 ETH)

### Visual
- Clean background (no notifications cluttering)
- Good lighting (no dark mode unless it's feature)
- Consistent color scheme with brand
- Readable text (not too small)

### Technical
- Export as PNG (better quality than JPG for UI)
- File size: <500KB each
- No alpha channel issues
- Test on actual device before submission

## Tools for Screenshots

### Mobile
- iOS: Built-in screenshot + Preview app for annotations
- Android: Built-in screenshot or scrcpy for desktop capture
- Browser DevTools: Device emulation mode

### Desktop
- Chrome DevTools: Full page capture
- CleanShot X (Mac)
- ShareX (Windows)

### Video
- Loom (quick demos)
- Screen Studio (polished)
- OBS Studio (free, full control)

### Editing
- Figma (design mockups)
- Canva (social media assets)
- Descript (video editing + captions)
