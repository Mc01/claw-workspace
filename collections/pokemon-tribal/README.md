## Design Language

| Principle | Implementation |
|-----------|---------------|
| **Silhouette-first** | All spirits recognizable as pure black outlines |
| **Pokemon simplicity** | Clean geometric shapes, no gradients, minimal detail |
| **African tribal patterns** | Zigzag, diamond, dotwork, spiral, stripe motifs |
| **Earth-tone palette** | Bone, ochre, clay red, forest green, storm gray |
| **Side profile** | All base spirits face right, consistent perspective |
